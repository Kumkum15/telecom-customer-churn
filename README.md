# telecom-customer-churn
Telecom Customer Churn Prediction project using multiple ML models including Logistic Regression, Random Forest, and XGBoost. Compared models using Accuracy, Recall, F1-score, and ROC-AUC, and selected Logistic Regression for its strong performance and interpretability.

This project predicts customer churn for a telecom company using machine learning techniques.

 ## Project Overview

Customer churn is a major challenge in the telecom industry. This project builds and evaluates multiple classification models to predict whether a customer will churn.

## Models Implemented

Logistic Regression

Random Forest

XGBoost

Gradient Boosting

Decision Tree

SVC

Naive Bayes

KNN

## Evaluation Metrics

Models were evaluated using:

Accuracy

Recall (Churn)

F1-Score (Churn)

ROC-AUC

Logistic Regression was selected as the final model due to:

Highest ROC-AUC score

Strong overall performance

Business interpretability
